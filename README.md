# FINVEST

**Team Bandwagons**

**Tagline:** Feel the risk, master the outcome.

Live: https://finvest-team-bandwagons.vercel.app/

## Problem We Are Solving

Many first-time and young investors avoid investing because risk feels confusing and scary. Most platforms focus on transactions, not confidence-building.

FINVEST solves this by helping users understand their risk mindset before they invest through simulations, quizzes, and guided learning.

## What FINVEST Does

- Converts fear of loss into measurable risk insights.
- Gives users a safe environment to test investment scenarios.
- Recommends learning-oriented paths based on behavior and quiz outcomes.
- Tracks user progress with profile-based dashboards and certificates.
- Adds smart assistant support for finance learning (AI chat).

## Core Features

- Risk profiling quiz and behavioral scoring
- Investor persona clustering and allocation suggestions
- Interactive market charts and visual insights
- Risk sandbox with scenario-based experimentation
- Supabase authentication and user profile management
- Optional AI-powered finance chat
- Optional blockchain certificate/badge support

## Tech Stack

- **Frontend:** React 19, Vite 8, React Router, Tailwind CSS 4, Framer Motion
- **Charts & Visuals:** Recharts, Lightweight Charts
- **Backend Services:** Supabase (Auth + PostgreSQL)
- **Serverless APIs:** Node.js handlers (chat and market routes)
- **Blockchain (optional):** Hardhat, ethers.js

## Quick Setup (Frontend)

```bash
cd FRONTEND
npm install
npm run dev
```

The app runs on the local Vite URL (usually `http://localhost:5173`).

## Note

FINVEST is an educational platform and does not provide personalized financial advice.
